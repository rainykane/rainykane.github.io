为什么开始写博客

九月 27, 2019

背景

大约两年前，习惯于在各大平台浏览博客学习技术，解决疑惑的我在某一天惊觉，“分享才是最好的学习方式”，仅仅只是记住了，或者会用了，还远远不够，更别说事实上我所谓的学习多半只是单纯的CTRL + C/V了。

分享的好处显而易见

  • 重新对知识进行系统的整理，加深印象
  • 在整理的过程中可能有新的收获

于是，写博客这件事就成了我一个不为人知的执念。

燃鹅，两年过去了！！你大爷还是你大爷，不对，你执念还是你执念ヽ(ｏ`皿′ｏ)ﾉ

趁着刚跟着Keep做完几组“减肥操”，鸡血效力还没过，在此立下Flag：

  • 关于写博客这件事既然开始了就决不轻易弃坑！
  • 停更不得超过两周！

写博客对不同的人的意义不同，对我自身来讲最迫切的则是希望开此坑能根治我的懒癌。最近懒癌作祟，已经丢掉了不少的东西了，再继续后果会很严重。再说作为一个已经过法定结婚年龄的老年人来说，理当成为核心竞争力的重要属性已经不再是“成绩好，上进心”，而是执行力强和高度自律。

现在的年轻人，有想法的人很多很多，但能将想法通过实践去检验的寥寥无几。现在的年轻人，要完成的daydayup目标很多很多，但能坚持到最后的没有1%。

写博客，则是我实现转变的第一步。flag都已经立下了，就往前冲把~

博客的内容

为博客的内容事先进行严格的分类我认为非常有必要。以下做分别的介绍：

  • Experience: 技术上经验的总结，文中技术上的观点尽量通过code来进行验证

  • Thinking: 技术除外的一些思考

  • Life: 生活的感受，个人规划或琐事记录等等……

  • Share: 此类目下的博文为纯分享，安利(•౪• )

    欢迎监督或共勉。
