普通键盘连接Mac的键位对应

Option键: Alt

Command键：win键

常用快捷键

cmd + q ：关闭窗口

cmd + m ： 最小化窗口

cmd + ctrl + F：窗口最大化

cmd + space : 切换输入法

cmd + tab : 窗口切换

cmd + <- ：网页回退

cmd + -> : 网页前进

cmd + backspace: 将文件移动至废纸篓

cmd + shift + 4: 截图

选中文件，回车 : 重命名

cmd + space：搜索（文件，应用啥的都能搜，巨好用）

一些Windows常用功能在Mac上的对应

Dos

  • 终端(Terminal)
  • 通过cmd + space唤起搜索，输入ter快速进入终端

文件系统管理

  • “我的电脑”对应Mac的“访达”（Finder）
  • 没有所谓的C,D,E,F分盘，文件系统和linux一样

输入大写字母

  • 长按“Caps lock”按钮为锁定“大写输入模式”,单按一次切换不了大写
  • 通过shift + x 来输入大写
