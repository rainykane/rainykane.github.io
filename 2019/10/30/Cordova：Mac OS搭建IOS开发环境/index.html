Cordova：Mac OS搭建IOS开发环境

十月 30, 2019
  • Cordova
  • Mac OS

安装依赖

Xcode

  • 需要先有一个Apple Id,有用过Iphone等苹果的电子产品的话都会有，可以直接登录，Apple Id是互通的，没有的话就按提示创建一个新的
  • 在App store中，搜索 Xcode 并安装下载（大概2G左右，可以先下着安装其他环境）

Node

  • 在官网下载macOS安装包（.pkg），按提示安装
  • http://nodejs.cn/download/

Cordova

  • 直接npm全局安装：npm install -g cordova

配置开发环境

安装完cordova之后就可以正常操作了

cordova create ${projectName}    # 创建cordova项目
cordova platforms add ios        # 添加ios平台
cordova emulate ios              # 在虚拟机上运行项目

注：此时如果希望通过cordova build ios 或 cordova run ios打出来测试包，在真机上预览测试的话需要在Xcode中对项目的证书和开发团队（需要签名认证）进行配置，否则会出现以下报错。
